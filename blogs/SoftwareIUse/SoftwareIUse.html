Software I use

plotting:

gnuplot and if very complicated python

text editing:

vim, with "text editing" I basically mean all files. E.g C++, python, sh,..

music:

ncmpcpp with mpd as backend as player, I also use yt-dlp to download songs from youtube,soundcloud,.. if I can't find a torrent. I then use a script to tag the song correctly and picard to get the cover art (or add it myself)

Document viewing:

Images:

sxiv

PDFs:

zathura

CIF:

There aren't many open source cif file viewers, happily jmol exists. A good alternative is XcrySDen, this is also open source but a bit hard to install if you aren't on a debian based distro.
