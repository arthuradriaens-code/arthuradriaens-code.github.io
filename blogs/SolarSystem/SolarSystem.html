Building a solar system in Unity

Building planets

We'll first start out by actually building planets, the plan is to be able to create a sphere and (ideally) map a real world photo of the planet on the sphere. To have good control over the amount of gridlines on the sphere we won't use a conventional sphere as shown to the right of this text as it has more detail towards the poles than in the middle. To get a sphere with equally spaced gridlines we'll inflate a cube as shown below.

Scripts

Now that we know what to do, we'll start actually doing. First we'll create 2 C# scripts in unity:

  • Planet.cs
  • TerrainFace.cs
The idea is that the planet script will be responsible for creating 6 terrain faces (cube) telling each of them which direction they're facing and each terrainface will then construct their own mesh.
